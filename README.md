# Travel Insurance Prediction using Machine Learning

## Project Overview
This project focuses on predicting whether a customer will purchase travel insurance using machine learning techniques. It analyzes customer demographics, travel behavior, and financial details to build predictive models.

---

## Dataset Information
The dataset contains 1987 records with 10 features, including:

- Age
- Employment Type
- GraduateOrNot
- AnnualIncome
- FamilyMembers
- ChronicDiseases
- FrequentFlyer
- EverTravelledAbroad
- Target variable: TravelInsurance (0 = No, 1 = Yes)

## Key Insights
- Frequent travelers are more likely to buy insurance
- Customers who travel abroad tend to purchase insurance
- Income level influences insurance decisions
- Employment type affects purchasing behavior
